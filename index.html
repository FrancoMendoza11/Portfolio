2048 - Juego de Estrategia Numérica

Ver repositorio
Desarrollo Java

Versión personalizada del clásico juego "2048", diseñado con una interfaz visual intuitiva y una mecánica de juego fluida. El objetivo es combinar números hasta alcanzar el valor 2048, utilizando movimientos estratégicos dentro de una grilla dinámica de 4×4.

Funcionalidades destacadas:

  • Interfaz gráfica moderna con Java Swing
  • Movimiento y combinación automática de fichas
  • Generación aleatoria de nuevos valores (2 o 4)
  • Sistema de puntaje acumulativo y registro de récords
